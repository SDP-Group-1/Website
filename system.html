Oli detail

Straight:

A straight-line movement done by setting the velocity of the motor of the wheels to positive values.

Turning:

The TurtleBot3 is a differential wheeled robot, this allows the OLI to gets the current location and rotation of the robot then computes an angle and finally rotates the robot using that angle until it is facing the target.
